Methods

  • Recordings

    • ClassificationAdd

      no implementation for this version
      Add a classification to a recording.
      Request:
      Resource:
      http://api.write-math.com/{version}/recordings/{recording_id}/classification
      URI Parameters:
      Parameter Type Optional Description
      version Version Api version
      recording_id integer Recording identifier
      Query String Parameters:
      Parameter Type Optional Description
      access_token string O-Auth access token which identifies and authenticates the user
      Body:
      {
      "classification": Classification Classification which is to be added
      }
      Response:
      Body:
      {
      "User": User
      }
      Codes:
      Code Message Description
      403 Forbidden User can not be adding to this customer
      404 User not found Customer does not exists
      Sample:
      Request:
      POST /v1.0/recordings/42/classification?access_token=my_access_token HTTP/1.1 Content-Type: application/json; charset=utf-8 { "classification": { "id": 42, "recording_id": 42, "symbol_id": 42, "user_id": 42 } }
      Response:
      HTTP/1.1 200 OK Content-Type: application/json; charset=utf-8 { "User": { "display_name": "my_display_name", "handedness": "my_handedness", "id": 42, "language": "my_language" } }
